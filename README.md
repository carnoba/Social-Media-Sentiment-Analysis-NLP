A robust, professional-grade NLP pipeline designed to mine and analyze Indian political discourse across **Twitter and Reddit**. This system preprocesses massive text datasets, implements TF-IDF vectorization, and achieves an **81.21% classification accuracy** to deliver deep brand and political perception mining.

## 🚀 The NLP Lifecycle

This project demonstrates a complete textual intelligence workflow:
1. **Raw Scrape Ingestion**: Processes large datasets from diverse social media platforms.
2. **Text Preprocessing**: Advanced cleaning including lemmatization, stop-word removal, and noise reduction.
3. **Vectorization**: TF-IDF (Term Frequency-Inverse Document Frequency) conversion for mathematical modeling.
4. **Classification**: High-performance Logistic Regression engine for sentiment prediction.
5. **Insights Engine**: Time-series perception tracking and aesthetic WordCloud generation.

## ✨ Elite Features

- **Platform-Agnostic Extraction**: Handles both Reddit and Twitter data structures seamlessly.
